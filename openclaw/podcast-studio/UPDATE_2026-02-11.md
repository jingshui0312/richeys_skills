## 新增功能

### 1. 智能脚本框架生成器（ScriptGenerator）

**位置：** `/root/clawd/skills/podcast-studio/scripts/script_generator.py`

**功能：**
- 根据主题和收集到的资料特征，动态生成合适的脚本框架
- 自动分析资料复杂度、内容类型、目标受众等特征
- 根据时长和风格调整脚本结构
- 为每个段落生成具体的创作指导

**框架生成逻辑：**
```
输入：主题 + 研究资料 + 风格 + 时长
  ↓
特征分析（复杂度、内容类型、目标受众、关键主题）
  ↓
框架调整（根据时长和特征）
  ↓
段落指导生成（为每个段落添加具体创作提示）
  ↓
输出：结构化脚本框架
```

**核心方法：**
- `generate_framework()` - 主生成方法
- `_analyze_research_features()` - 特征分析
- `_extract_themes()` - 主题提取
- `_adjust_framework()` - 框架调整
- `_add_section_guides()` - 添加创作指导

### 2. 脚本质量检查器（ScriptChecker）

**位置：** `/root/clawd/skills/podcast-studio/scripts/script_generator.py`

**功能：**
- 自动检查脚本通畅性和口语化程度
- 提供详细的评分和改进建议
- 识别常见问题（长句、书面语、缺少过渡词等）

**检查维度：**

**通畅性检查（Fluency）：**
- 句子长度分析（平均句长、超长句子检测）
- 过渡词使用检查
- 用词重复度检测

**口语化检查（Colloquialism）：**
- 书面语识别（"因此，"、"然而，"等正式表达）
- 口语化表达检查（"然后"、"但是"等自然连接词）
- 听众互动度检查（"我们"、"大家"、"你"等互动词汇）

**评分标准：**
- 90-100分：优秀 - 可直接用于音频生成
- 75-89分：良好 - 建议微调
- 60-74分：一般 - 建议优化
- 0-59分：较差 - 需要大幅调整

**核心方法：**
- `check_script()` - 主检查方法
- `_check_fluency()` - 通畅性检查
- `_check_colloquialism()` - 口语化检查
- `_generate_suggestions()` - 生成改进建议

### 3. 脚本优化器（ScriptOptimizer）

**位置：** `/root/clawd/skills/podcast-studio/scripts/script_generator.py`

**功能：**
- 根据检查结果自动优化脚本
- 应用书面语→口语化替换规则
- 自动拆分长句
- 增加过渡词和互动表达

**优化规则：**
```python
书面语替换：
"因此，" → "所以，"
"然而，" → "但是，"
"综上所述，" → "简单来说，"
"由此可见，" → "可以看出来，"
```

**优化逻辑：**
1. 替换书面语为口语化表达
2. 拆分过长句子（在逗号处分句）
3. 增加段落间过渡词
4. 增加听众互动表达

**核心方法：**
- `optimize_script()` - 主优化方法
- `_optimize_section()` - 单段落优化
- `_add_transitions()` - 添加过渡词
